05 / 设计结论
第一版聚焦 个人本地模式、多项目、多 Agent Profile、自定义 workflow、manual gate、worktree、工程反馈、记忆建议。
后续扩展 团队模式、权限协作、云端 Agent 调度、向量检索/RAG、更完整的 Capability Marketplace。
