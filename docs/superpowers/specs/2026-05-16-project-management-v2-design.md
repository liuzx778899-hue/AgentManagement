# 项目管理 V2 设计方案

日期：2026-05-16
状态：设计方案
效果参考：`mockups/project-management-v2.html`

## 目标

将当前 ProjectManagement 重构为"看板首页 + 项目详情面板"两层结构。导入/新建改为紧凑按钮，Issues/PR/CI 移入项目详情内部。

## 问题

当前首页直接展示"导入项目""新建项目"两个大卡片 + 右侧接入标准/项目列表/Issues/PR/CI——层级混乱，信息过载。

## 组件拆分

| 组件 | 职责 |
|------|------|
| `ProjectManagement.tsx` | **重写** — 页面容器：统计行 + 看板网格 + 近期任务 + 详情模态 |
| `ProjectCard.tsx` | **新增** — 项目看板卡片（名称/技术栈/进度条/状态统计） |
| `ProjectDetail.tsx` | **新增** — 项目详情面板（tab: Issues/PR/CI/工作流与角色/设置） |
| `IssueList.tsx` | 保持不变，重用于详情面板内 |
| `PullRequestList.tsx` | 保持不变，重用于详情面板内 |
| `CiPipelinePanel.tsx` | 保持不变，重用于详情面板内 |
| `ExistingProjectImport.tsx` | 保持不变，触发方式改为顶部按钮 + 弹窗 |
| `NewProjectWizard.tsx` | 保持不变，触发方式改为顶部按钮 + 弹窗 |

## 交互要点

1. 进入页面默认看板视图
2. 顶部两个按钮："导入项目""新建项目"
3. 点击卡片 → 弹出详情面板（覆盖层 + 内部面板）
4. 点击覆盖层空白处/关闭按钮 → 关闭详情
5. 详情内 tab 切换 Issues/PR/CI/CD/工作流与角色/设置
6. 导入/新建点击后弹出覆盖层（复用 ExistingProjectImport/NewProjectWizard）
