# 项目管理 V3 + 项目工作区 设计方案

日期：2026-05-16
状态：设计方案
效果参考：
- `mockups/project-management-full.html` — 看板 + 导入/新建 + 人工决策 + 进入工作区
- `mockups/project-workspace-v1.html` — 项目工作区（流程 + Agent 对话 + 文档索引）

## 目标

1. **项目管理 V3**：替换当前 P0 旧版 ProjectManagement 为看板式首页，含导入/新建弹窗 + 人工决策面板
2. **项目工作区**：新增进入项目后的完整开发界面（工作流执行 + Agent 对话 + 文档索引）

## 架构

两层页面路由：
- 首页 = 项目看板（ProjectManagement）
- 点击项目卡片 → 全页面切换到项目工作区（ProjectWorkspace）
- 工作区内可触发人工决策面板（覆盖层）
- 导入/新建通过覆盖层弹窗触发

## 功能清单

### 项目看板首页

```
┌─ 顶部栏 ───────────────────────────────────────────┐
│  项目管理                       [导入项目] [新建项目] │
├─────────────────────────────────────────────────────┤

├─────────────────────────────────────────────────────┤
│  ┌─ Agent Management ───────┐  ┌─ Internal-CLI ──────┐ │
│  │ main · Vite+React+TS  │  │ develop · Go+Cobra   │ │ ← 项目卡片网格

│  │ 1运 1决 3完           │  │ 1运 0决 1完          │ │

│  近期任务                                           │

```

**交互：**
- 顶部"导入项目" → 弹出导入面板（3步：扫描→检测→确认）
- 顶部"新建项目" → 弹出新建面板（3步向导：基本信息→角色&流程绑定→确认）
- 点击项目卡片 → 切换到项目工作区全页面
- 项目卡片上有 Gate 待决策时，右上角显示橙色徽章，点击打开人工决策面板

### 人工决策面板

覆盖层弹窗，不离开看板：
- 左侧：证据面板（Diff / 测试 / 预览 / 日志 四个 tab）
- 右侧：决策面板（摘要 + 记忆保存 + 4 操作按钮：同意/重跑/改派/终止）

### 导入项目

3 步流程（覆盖层弹窗）：
1. 扫描：输入路径 + 选择文件夹 + 开始检测
2. 检测结果：技术栈/命令/检测到的角色（可编辑）
3. 确认并导入

### 新建项目

3 步向导（覆盖层弹窗）：
1. 基本信息：名称/路径/分支/技术栈 + 命令配置 + 风险评估 + 绑定流程模板
2. 角色 & 流程绑定（预留）
3. 确认

### 项目工作区

全页面视图，双栏布局：
- **左侧**：概览统计(4指标) + 当前任务卡片 + 工作流执行进度 stepper + 协同文件文档索引 + Agent 运行日志
- **右侧**：Agent 列表(5个Agent切换) + 对话面板 + 输入框
- **顶部**：「← 返回项目管理」按钮 + 项目名 + 快捷按钮(人工决策/Issues/CI/CD/设置)

## 组件拆分

| 文件 | 类型 | 职责 |
