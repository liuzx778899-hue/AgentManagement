# 设计文档

本目录保存 AgentManagement 已确认的 UI/UX 设计资料。

## 当前设计依据

- [[design-overview|设计总览]]：产品级设计总览。
- [[page-spec|页面规格]]：页面级布局和交互规范。
- [[design-tokens|设计 Token]]：颜色、间距、字体和可复用视觉变量。
- [[design-gap-fix-plan|设计差距修复计划]]：设计稿与实现之间的修复计划。
- [[design-vs-implementation-gap|设计与实现差距]]：差距审查记录。
- [[assets/current/README|当前设计资产索引]]：当前可参考的页面截图和图片资产。

## 资源目录

- `mockups/current/`：当前用于开发实现的 mockup。
- `assets/current/`：当前用于开发实现的截图和图片参考。
- `mockups/archive/`：旧版 mockup、根目录遗留 mockup、公开预览导出和探索稿。
- `assets/archive/`：旧版截图或图片参考。

## 使用规则

- 开发实现默认只参考 `mockups/current/` 和 `assets/current/`，除非交接文档明确指定其他资料。
- 旧版 mockup 保留用于追溯，不应覆盖当前设计规范。
- 如果某个生成设计被确认采用，需要移动到 `mockups/current/` 或 `assets/current/`，并同步更新本 README 或 [[page-spec|页面规格]]。
