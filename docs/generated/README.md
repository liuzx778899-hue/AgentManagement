# 生成资料

本目录保存 AI 工具、Skills、Superpowers、Codex、Claude Code 或类似 Agent 生成的过程文档。

这些文件是工作材料，默认不是当前开发的最终依据。被确认采用的内容，需要整理到 `docs/product/`、`docs/architecture/`、`docs/design/` 或 `docs/handoff/`。

## 使用规则

- 生成的计划、规格说明和状态记录统一保存在这里。
- 不要让实现 Agent 把本目录当作第一信息源。
- 被确认采用的内容，需要提升到正式文档目录。
- 过期但仍有追溯价值的生成资料，可以移动到 `docs/archive/`。
